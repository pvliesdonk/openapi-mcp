# Installation

## From PyPI

```bash
pip install pvliesdonk-openapi-mcp
```

## From Docker

```bash
docker pull ghcr.io/pvliesdonk/openapi-mcp:latest
```

## From source

```bash
git clone https://github.com/pvliesdonk/openapi-mcp
cd openapi-mcp
uv sync --all-extras --all-groups
```
